# Code of Conduct

Cairn adopts the [Contributor Covenant 2.1][cc] as its code of conduct.

The short version: be respectful, assume good faith, focus on the
technical work, and disagree without being unpleasant. We do not have
patience for harassment, personal attacks, or behavior that pushes
contributors out.

## Reporting

If you see or experience behavior that violates this code, email
**cairn-conduct@kaldera.dev**. Reports are read by the maintainers and
treated confidentially. We will respond within five working days.

If the report concerns a maintainer, send it to a different maintainer
listed on the project's GitHub page.

## Enforcement

Maintainers may, at their discretion:

- Ask the participant to stop the behavior.
- Edit or remove offending content.
- Issue a temporary ban from project spaces.
- Issue a permanent ban for repeated or severe violations.

Decisions are documented in the project's private moderation log.

## Scope

This code applies to all project spaces: GitHub issues + PRs, the
discussion forum, any official chat channel, and project-affiliated
events online or in person. It also applies when an individual is
representing the project in public, e.g. via a project email address or
as an appointed maintainer.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
